UI Design

Beasts of the Underworld UI Project

The Beasts of the Underworld UI project was a UI Design task that I undertook while in graduate school. The purpose of this assignment was to give a foundation and exposure to creating menu system in Unreal Engine 4. This this assignment, I created a main menu, high scores menu, and a game HUD.

The Space Station

The Space Station is an example Web Design and UI Design that I developed when I was pursuing my certificate in Software Development at the Tech Academy in Portland, Oregon. The Tech Academy is a registered Technical Trade School where I studied and trained to be a Full-Stack Web Developer. This project focuses on front-end development and UI Design for a landing page for a company.

The Chupacabara Music Festival

The Chupacabra Music Festival website is an example of both my web design and UI Design skills to provide a unique webpage for a music festival.

Tulsa Summer Party

The Tulsa Summer Party Landing Page demonstrates an example of my ability both with UI design and my ability to incorporate JavaScript into a website. The counter counts down for the next July 4th event!
