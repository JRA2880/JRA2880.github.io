The Sea Fox Project

Overview

The Sea Fox Project is a game development project for Lockheed Martin.

The purpose of the game is test a submarine AI's ability to maneuver around land mines.

Role: Technical Designer

Team size: 14

Game Engine: Unreal Engine 4

Main Responsibilities and Duties:

  • Developed 1st Pass Menu UI and UI Logic
  • Created a Object Spawning Volume that randomly spawns objects in volume
  • Implemented a Player Death Collision box to test when player submarine runs into rocks



Random Spawning Object Volume

To help test the player submarine AI, it is necessar to have a method of spawning underwater mines in a volume so that the AI can be tested multiple times in various locations.
