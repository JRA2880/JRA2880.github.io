Game Design Portfolio

Sabotage

Sabotage is an RTS game that combines tycoon mechanics to create a unique form of gameplay that challenges the player, and provides multiple avenues to victory. As with any great game design, the starting focal-point for this game design was the target audience. For this design, the target audience was honed in on the specific personality type of INTJ, the Architect personality type. This target audience member enjoys mental challenges that other players shun from. An RTS game that combines tycoon mechanics, resource management, corporate sabotage is a challenge that an INTJ would enjoy.
