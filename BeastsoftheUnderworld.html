UI Design

Beasts of the Underworld UI Project

Overview

Beasts of the Underworld is an example UI project.

This project provides an example of creating UI systems in UE4

Role: Technical Designer

Team size: 1

Game Engine: Unreal Engine 4

Main Responsibilities and Duties:

  • Developed the Main Menu, High Scores Menu, and Game HUD
  • Prototyped the animation of menu buttons
  • Created Blueprint Scripts to transition from Menu to Menu
  • Designed the Game HUD to allow player to pause, determine time remaining, and determine amount of ammo used and reload
  • Created Blueprint functions to calculate time remaing and ammo remaining

Menu System Design and Game HUD Design

The main focus of this project was to design a main menu, high scores menu, and a Game HUD. The following was achieved with this project:

  • Utilized specialized Main Menu and High Scores Menu Art to provide a unique look and feel to the project.
  • The animation of the New Game and High Score buttons to provide a unique hover effect when the player selects a button.
  • Game HUD Design for a Pause, Timer, and Ammo UI for the player.
  • Binding of a Timer Function for The Timer Bar
  • Binding of an Ammo Count Function for the Ammo Bar

Menu Blueprints and Game HUD Desgin

To help create a template in which multiple classes can be created, a player controller, and interfaces was developed in Unreal visual scripting to allow this prototype the ability to add multiple classes without having to be redundant in code.
